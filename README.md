# Online Registration System

## Description
A web-based online registration system built using PHP and MySQL.
This application allows users to register online, receive a registration code,
and check their graduation status. An admin dashboard is provided to manage
student data and determine graduation results.
This project is created for learning and portfolio purposes.

## Features

### User Features
- Online registration form
- Automatic generation of registration code after form submission
- Check graduation status using registration code
- Logout functionality

### Admin Features
- Admin dashboard
- View total number of registered students
- View and manage student data
- Manage student address data
- Manage parent/guardian data
- Set student graduation status (Pass / Not Pass)
- Logout functionality

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## How to Run Locally (Laragon)
1. Install Laragon
2. Start Apache and MySQL from Laragon
3. Copy the project folder into `C:\laragon\www`
4. Import the database file (.sql) using phpMyAdmin
5. Configure database connection in the configuration file
6. Open the project in your browser via `http://localhost/project-folder-name`
